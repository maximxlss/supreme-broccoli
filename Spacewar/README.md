# Spacewar

## Dependencies

- Python 3
- Numpy - `pip install numpy`
- Pygame - `pip install pygame`

## Decription

Simple unfinished game inspired by Spacewar in Python with tk. Run [game.py](./game.py).

Controls: Arrows

Was just an experiment with gravity physics
